# Electrospray-Propulsion-Codes
This Repo is all about electric propulsion specifically to electrospray. We are calculating the thrust produce and using electric field solver to solve for the 3D electric field profile. So that one can optimize the geometry of the array and can come up with the optimum design in very fast and itreative manner rather going through a slow cycle of experimenting and then designing.
Currently, the Repo is only about Electrospray(Liquid metal electric Propulsion), but soon will be adding FEEP as well
